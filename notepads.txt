## 웹 초보자를 위한 명언 랜덤 생성기 (Random Quote Generator) PRD

### 1. 개요

이 프로젝트는 웹 개발 초보자들이 HTML, CSS, JavaScript를 사용하여 간단한 웹사이트를 만들어보는 경험을 제공하기 위한 **명언 랜덤 생성기** 웹사이트입니다. 사용자가 버튼을 클릭하면 새로운 명언과 작가가 랜덤으로 표시됩니다.

---

### 2. 목표

* HTML, CSS, JavaScript의 기본적인 사용법 익히기
* 웹 페이지 레이아웃 구성 및 스타일링 경험
* JavaScript를 이용한 동적인 웹 페이지 구현
* 간단한 웹 프로젝트 완성 경험

---

### 3. 페르소나

**이름:** 김민아 (가명)
**나이:** 29세
**직업:** 프리랜서 디자이너
**상황:** 최근 번아웃과 우울감으로 인해 작업에 집중하기 어렵고, 매일 아침 동기 부여를 얻는 데 어려움을 겪고 있습니다. 유튜브에서 동기 부여 영상을 찾아보거나, 짧은 글귀를 읽으며 힘을 얻으려 하지만, 매번 찾아보는 것이 번거롭다고 느낍니다.

**니즈:**
* **간단하고 즉각적인 동기 부여:** 복잡한 내용보다 짧고 임팩트 있는 메시지를 원합니다.
* **손쉬운 접근성:** 언제든 빠르게 명언을 볼 수 있는 방법을 선호합니다.
* **새로운 자극:** 매번 다른 명언을 통해 신선한 영감을 얻고 싶어 합니다.

**이 프로젝트가 김민아에게 주는 가치:**
이 웹사이트는 김민아 씨가 웹 개발이라는 새로운 분야에 도전하면서, 동시에 매일 아침 새로운 명언을 통해 작은 용기와 영감을 얻을 수 있도록 돕습니다. 직접 만든 웹사이트를 통해 스스로에게 긍정적인 메시지를 전달하며 성취감도 느낄 수 있을 것입니다.

---

### 4. 기능

* **명언 표시:** 웹 페이지 중앙에 랜덤으로 선택된 명언이 표시됩니다.
* **작가 표시:** 명언 아래에 해당 명언의 작가가 표시됩니다.
* **새 명언 버튼:** "새 명언 보기" 버튼을 클릭하면 새로운 명언과 작가가 나타납니다.
* **초기 로딩:** 웹 페이지 로딩 시 자동으로 하나의 명언이 표시됩니다.

---

### 5. 디자인 (간단한 초안)

* **전체적인 분위기:** 깔끔하고 심플한 디자인
* **배경:** 단색 또는 은은한 그라데이션
* **명언 영역:** 명언과 작가를 감싸는 상자 (카드 형태)
    * **명언 폰트:** 읽기 쉽고 적당한 크기 (예: `font-size: 1.5em;`)
    * **작가 폰트:** 명언보다 작게 (예: `font-size: 1em;`)
* **버튼:** 눈에 잘 띄는 색상과 텍스트 (예: "새 명언 보기")

---

### 6. 기술 스택

* **HTML5:** 웹 페이지의 구조를 만듭니다. (명언, 작가 표시 영역, 버튼 등)
* **CSS3:** 웹 페이지를 예쁘게 꾸밉니다. (폰트, 색상, 레이아웃 등)
* **JavaScript (ES6):** 명언을 랜덤으로 선택하고, 버튼 클릭 이벤트를 처리하며, 웹 페이지에 동적으로 내용을 표시합니다.

---

### 7. 테스트 (간단한 확인)

* 웹 페이지가 정상적으로 로드되는지 확인합니다.
* "새 명언 보기" 버튼을 클릭했을 때 명언과 작가가 바뀌는지 확인합니다.
* 명언과 작가의 내용이 깨지지 않고 잘 표시되는지 확인합니다.
